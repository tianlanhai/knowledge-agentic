## 🎨 设计理念

本项目采用 **UI/UX Pro Max** 设计理念（参考 https://ui.cod.ndjp.net/），融合现代UI/UX最佳实践，打造专业、优雅、易用的界面体验。

### 核心设计原则

- **玻璃拟态（Glassmorphism）**：采用半透明、模糊背景和边框光效，营造现代感和层次感
- **深色模式（Dark Mode）**：基于深色背景（#0f172a）和浅色文字，提供高对比度和舒适的视觉体验
- **Bento Grid布局**：采用网格化卡片布局，响应式排列，信息层次清晰
- **渐变美学**：蓝紫渐变（#667eea → #764ba2）作为主色调，传递专业和创新感
- **无障碍设计（Accessibility）**：符合WCAG 2.1标准，确保所有用户都能便捷使用

### 设计系统

#### 配色方案

| 色系 | 颜色值 | 用途说明 |
|-----|-------|---------|
| **主色调** | #667eea → #764ba2 | 蓝紫渐变，用于按钮、高亮元素、品牌标识 |
| **背景色** | #0f172a | 深色背景，提供舒适的视觉基础 |
| **卡片背景** | #1e293b | 内容容器背景，与主背景形成层次 |
| **主要文字** | #f1f5f9 | 标题和重要信息，高对比度 |
| **次要文字** | #cbd5e1 | 正文内容，清晰易读 |
| **辅助文字** | #94a3b8 | 提示信息，弱化显示 |
| **玻璃态** | rgba(255, 255, 255, 0.05) | 半透明背景，营造层次感 |

#### 字体配对

| 用途 | 字体 | 特性 |
|-----|------|------|
| **标题** | Inter | 现代、清晰、可读性强 |
| **正文** | Inter | 统一风格，保持一致性 |
| **代码** | JetBrains Mono | 等宽字体，适合代码展示 |

#### 间距系统

- **基础单位**：4px
- **间距规范**：8px（sm）、16px（md）、24px（lg）、32px（xl）、48px（2xl）
- **组件间距**：12px（内部）、16px（组件间）、24px（区块间）

#### 阴影与光效

- **卡片阴影**：0 8px 32px rgba(0, 0, 0, 0.4)
- **光晕效果**：0 0 20px rgba(102, 126, 234, 0.4)
- **玻璃态阴影**：0 8px 32px 0 rgba(0, 0, 0, 0.3)

#### 动画与过渡

- **过渡时间**：150ms（快）、300ms（中）、500ms（慢）
- **缓动函数**：cubic-bezier(0.4, 0, 0.2, 1)
- **常用动画**：淡入（fadeIn）、滑入（slideIn）、脉冲（pulse）

## 📊 项目概览

### 技术栈
- **后端**：FastAPI + LangChain + Ollama + ChromaDB
- **前端**：React 18.3.1 + Vite 7+ + pnpm + TypeScript + Ant Design 6+
- **AI**：Ollama (Llama 3 / DeepSeek-R1)
- **向量数据库**：ChromaDB
- **测试**：pytest (后端) + Vitest (前端)

### 界面设计特性
- ✅ **现代UI风格**：玻璃拟态、深色模式、Bento Grid布局
- ✅ **响应式设计**：支持桌面端和移动端
- ✅ **动画过渡**：流畅的交互体验
- ✅ **无障碍设计**：符合WCAG 2.1标准
- ✅ **主题定制**：基于Tailwind CSS的主题系统

### 核心功能
- ✅ 多源知识摄入（文件、网页、数据库）
- ✅ 语义检索与溯源
- ✅ 智能问答与多轮对话
- ✅ 文档总结与对比分析
- ✅ 智能体任务执行
- ✅ **敏感信息过滤**（手机号、邮箱自动脱敏）
- ✅ GPU 加速支持

### 测试覆盖率目标
- **后端整体覆盖率**：> 95%
- **前端整体覆盖率**：> 80%
- **核心业务逻辑**：> 98%

### Q6: 什么是UI/UX Pro Max设计理念？
**A**: UI/UX Pro Max是一个专业级设计智能数据库，提供57种UI风格、95种配色方案、56种字体配对和98条UX准则。本项目采用了其中的玻璃拟态、深色模式、Bento Grid布局等现代设计元素。

### Q7: 如何自定义界面主题？
**A**: 前端使用Tailwind CSS进行样式管理，可以在 `tailwind.config.js` 中自定义配色方案和间距系统。

### Q8: 界面支持哪些浏览器？
